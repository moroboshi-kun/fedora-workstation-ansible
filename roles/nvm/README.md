# nvm

Installs NVM and a pinned Node.js version.

## Variables

| Variable | Default | Description |
|---|---|---|
| `nvm_version` | `v0.40.4` | NVM release to install |
| `nvm_dir` | `~/.config/nvm` | NVM installation directory |
| `nvm_node_version` | `24.14.1` | Node.js version to install via NVM |
